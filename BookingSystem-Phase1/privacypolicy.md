# Privacy Policy

## Introduction
This booking system respects the privacy of its users and complies with the General Data Protection Regulation (GDPR). All personal data is processed lawfully, fairly and transparently.

## Data Collected
We may collect the following personal data during registration and use:
- Email
- Password
- Birthdate
- Role

## Purpose of Data Collection
Personal data is collected for the purpose of:
- Creating and managing user accounts
- Allowing users to reserve resources
- Managing bookings within the system

## Data Visibility
Bookings may be visible to all users, but personal data such as emails are never displayed publicly. Only administrators can access certain personal data in the system.

## Data Security
The system implements security measures to protect user data, including:
- Secure login with hashed passwords
- Role-based access control
- Minimal personal data stored in logs

## User Rights
Users have the right to:
- Access their personal data
- Request correction of incorrect data
- Request deletion of their personal data
- Withdraw consent for processing at any time

## Data Retention
Personal data is stored only as long as necessary for the operation of the booking system and will be deleted when no longer needed.

## Contact
For questions regarding privacy or personal data processing, please contact the system administrator at [admin@example.com].
